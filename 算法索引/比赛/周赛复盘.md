### 1. 题目类型 + 知识点

都属于字符串的问题

- 字符串和数字的转换
- 子串和距离s[i]最近的前一个相同的字符是哪一个
- 子串和字符串hash
- 子串和动态规划，是第二题的进阶版





```java

/*

	最后一题的代码
	sum : 表示是s[i]为以s[i]结尾的子串提供个数。
	1. 如果前面没有提供了i + 1个 其中1个是他自己
	2. 如果前面最靠近它的是j, 提供了 i - j个
*/


public void solve() {
    int sum = 0, ans = 0;
    for (int i = 0; i < str.length; i++) {
        int c = str.charAt(i) - 'a';
        sum += i - p[c];
        p[c] = i;
        ans += sum;
    }
    return ans;
}

```
