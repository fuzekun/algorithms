# 牛客的动态规划


- [被3整除的子序列](https://ac.nowcoder.com/acm/problem/21302)
    
    子序列
    
    f[i] : 表示以i结尾的子序列可以被3整除
    
    f[i][j]:表示以i结尾，并且对3取余之后为j的个数。
    
    每一个以j结尾的，加上x之后，就变成了以(j + x) % 3结尾的。
    然后每一个(j + x) % 3在加上原本的(j + x) % 3就行了。
    
    f[i][(j + x) % 3] = f[i - 1][(j + x) % 3] + f[i - 1][j];
